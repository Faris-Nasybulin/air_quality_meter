Устройство измерения концентрации углекислого газа на основе датчика SenseAir S8 с 7 светодиодами для отображения информации и одной кнопкой для управления.

# Зависимости:
- pymodbus
- pyserial

# Хотелки к реализации:
+ считывать данные с сенсора S8
- писать лог измеренной концентрации в файл с указанием времени измерения. Если глобальное время не известно, то писать время от старта устройства, а после установки глобального времени - зафиксировать сдвиг для пересчета.
- отображать текущие показания сенсора S8 зажигая только 1 из 4 цветных светодиодов при превышении одного из порогов 400, 800, 1600, 3200 ppm.
- отображать текущие показания сенсора S8 используя 1 светодиод в режиме ШИМ для указания на один из 5 диапазонов измерений, самое значение отображать оставшимися 6 светодиодами:
	- менее 340 - указывать зашкаливание зажиганием всех светодиодов, первый диапазон
	- от 340 до 400 - первый диапазон, значение отображается как 400 - концентрация
	- от 400 до 1000 - второй диапазон, (концентрация - 400) / 10
	- от 1000 до 2000 - 3й, (концентрация - 1000) / 20
	- от 2000 до 5000 - 4й, (концентрация - 2000) / 50
	- 5000 и более - 5й, (концентрация - 5000) / 100
- зажечь все светодиоды
- выключить все светодиоды
- по короткому нажатию на кнопку переключаться между режимами отображения
- по длинному нажатию на кнопку останавливать скрипт и выключать систему
